# 🚀 Projeto Reobote Final

## 📌 Sobre o projeto
O **Projeto Reobote Final** é uma aplicação desenvolvida como projeto de conclusão, com o objetivo de aplicar na prática conhecimentos de desenvolvimento web, organização de código e construção de sistemas completos.

A aplicação simula um sistema real, integrando interface e lógica de funcionamento, com foco em usabilidade, estrutura e boas práticas de desenvolvimento.

---

## 🎯 Objetivo
- Consolidar conhecimentos adquiridos durante os estudos  
- Desenvolver uma aplicação funcional do zero  
- Praticar organização de código e versionamento com Git/GitHub  
- Criar um projeto para portfólio profissional
